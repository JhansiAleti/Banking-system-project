# Banking System Mini Project
## How to Compile:
$ make

## How to Run:
$ ./bank

## Features:
1. Create account.
2. Transaction history.
3. Withdraw amount.
4. Deposit amount.
5. Balance inquiry.
6. Transfer money.
7. Display all accounts.
8. Save account information in a file.
9. Quit the application.

## To Clean:
$ make clean
